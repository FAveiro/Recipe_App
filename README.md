# Recipe_App
Description: A recipe-filtering web application that lets users, with or without complex dietary need, find and bookmark recipes.

# Tecnologies used:

### Front-end:
* Tailwind
* React-router-dom
* axios
* react-cookie
* react-responsive-carousel
* formik && yup
* react-toastify
* react-responsive
* phosphor-icons/react
* react-loader-spinner

### Back-End:
* express
* axios
* cors
* dotenv
* bcrypt
* jsonwebtoken
* mongoose
